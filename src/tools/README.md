# src/tools/README.md

## Tools для аналитического агента

| Файл | Назначение |
|------|------------|
| `analytics.py` | Общая аналитика: сводки, тренды, сравнение периодов |
| `ads.py` | Рекламные кампании: ДРР, фильтры, аномалии |
| `sku.py` | Анализ по артикулам: маржинальность, убыточные |
| `visualization.py` | Графики и отчёты |
